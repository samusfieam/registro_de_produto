#Registro de Produtos 
Sistema de Registro de Produtos.



Tecnologias utilizadas:
Aplicação usando o desing patterns MVC.

Frontend:
	- MaterializeCss
	- Javascript
	- Angular 1.6
    
Backend:
	- NodeJS
	- MongoDB
	

Instalação do Projeto
1. Download do eclipe neon
2. git clone [https://github.com/samusfieam/registro_de_produto.git](https://github.com/samusfieam/registro_de_produto.git)
3. Download do MongoDb [https://www.mongodb.com](https://www.mongodb.com) com os arquivos de configuração.
4. Criar um banco de dados no mongoDB com o nome "produtodb".
5. Subir o MongoDb, usando o comando : mongod.
6. Instalar as dependências do projeto: npm install -d
7. Executar o projeto: npm run start.
